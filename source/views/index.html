• The Third Gift Games presents: Rolehaven - The larp companion app

  • Rolehaven is an initiative to create a universal app platform that can easily be used in any type of larp.
  • The project is open source and available at Github: backend.

  • Try it out!

  • You can try out the some versions of the app that were used on previous events at:
  • Blodsband Reloaded (Mad Max everything goes style). Works on: phones, desktop with Chrome.
  • Tales from the Sprawl (cyberpunk, Shadowrun). Works on: phones, desktop with Chrome.

  • Getting the latest updates

  • You can get semi-regular updates about the project at Facebook or Patreon.

  • Help out!

  • The project is kept alive by your donations. Any help, no matter how small, makes a difference and is appreciated!

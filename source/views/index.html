ROLETERMINAL

ROLETERMINAL

roleTerminal is a web application that acts as a in-game game/tool for live action roleplaying and roleplaying events.

It has been successfully used at several Swedish LARP events, such as Blodsband Reloaded and Ockulta Medborgarbyrån

You can try it out yourself here. The project is open source and the code is available at Github

How to

Shortcuts

Page up/down or ctrl+arrow up/down scrolls the view on desktops. Arrow up/down allows you to go through commands that you have previously used. Tab/double space auto-completes commands (see topic further down)

ctrl+u will turn on full screen on desktops. ctrl+d is a shortcut for the logout command (see topic further down)

Help

Input "help" to get tips, instructions and available commands. Pressing the tab button, or space twice on touch interfaces, has the same effect.

help

Input "help" followed by the name of the command to get instructions for a specific command

help login

Run command

You can either press the enter key or press the EXEC, short for execute, button on the top left to run commands.

Register account

Input "register" followed by your user name and press the Enter key. Note that your name can have a max length of 6 characters and/or numbers. You will be asked to choose a password and to repeat it. The password will not be shown on the screen while you type it.

register abrams

Logging in and out

Input "login" followed by your user name to log in. You will be asked for your password. When you are done you will get a message telling you that you are logged in.

login abrams

NOTE! You have to use “logout” command to log out of your account after using it on a public terminal. Input “logout” to log out. You can also use the shortcut ctrl+d

logout

Messages

Input in "msg"" followed by your message.

msg Hello, world!

Messages that are retrieved have more information than just the message itself. Starting from the left, you will see the time of when the message was sent, to which room it was sent, by whom and the actual message.

08:00 public> nava: Hello

Private message

Input "whisper"" followed by the user you want to send a private message to. You can also click on the user's name instead of manually typing the command.

whisper beta Hello!

History

Input “history” to retrieve old messages from all your rooms. You can specify how many messages you want to retrieve and if you want to retrieve messages from a specific room. Input “history public 100” to retrieve the 100 latest messages sent to the room “public.”

history

history public 100

Archives

Archives containt in-game documents written by organisers and/or players. Input "archives list" to see a list of available public documents. Input "archives" followed by the ID of the document you want to read.

archives list

archives 55GCZ12

World map

Input in "map on" to view the map. This zooms the map in on our current area. Input “map on overview” to show all the available points on the world map.

map on

map on overview

The cross icon indicates a stationary point, such as a settlement or base. The vertical line icon represents you. The horizontal line icon represents other users. The 3 horizontal lines icon represents your team members

Input "map locate" followed by a user name to locate that user on the map. The map will zoom in on the users location (if the user is found)

map locate mina

You can zoom in/out an drag the map view around with your touch interface. You can also use options "zoomin/zoomout" to do the same. Input "map zoomin" or "map zoomout"

map zoomin

map zoomout

Map info

You can read the full description for a point if you input "map info" followed by the number of the point. You can also click on a point to read its description.

map info 42

Expanding the map view

Clicking on EXPAND/COLLAPSE in the top menu when the map is open will epxand/collapse the map view.

EXPAND =>

COLLAPSE <=

Turning off the map

Input "map off" to turn off the map.

map off

Following and unfollowing rooms

Input "follow" followed by the name of the room. You will be asked to input a password. Leave it blank if the room has no password. You can follow as many rooms as you want. Messages sent to the rooms you follow will be shown in your feed. Input "unfollow" followed by the name of the room to unfollow a room

follow room21

unfollow room21

Switching rooms

Input "room" followed by the name of the room. This will now become your default room, the room that the messages you send will be sent to. You will still retrieve messages from all your followed rooms Your default room is shown on the bottom left, right above your input

room room21

User information

Input "whoami" to show you which user you are logged in as, the rooms you follow and other info

whoami

Auto-complete

You don't have to input all commands manually. Try typing "hi" and press tab (or double space on touch devices). That "hi" should have turned into "history" by itself.

If there are more than one command that starts on a letter all of them will be shown. Try typing in "r" and press tab (or double space on touch devices).

Create room

Input "create room" followed by the name of the room. You will be prompted to input a password. Leave it empty if you don't want to password protect your room

create room rr2

You can use "invite room" followed by user name and room name to invite a user to a room you are following. They can accept the invitation with "invitations"

invite room user1 rr2

invitations

Teams

You can create teams. Everyone who is part of the same team will automatically follow the same team room (called "team"). Your team members will also show up as unique icons on the map.

Input "create team" followed by the name of the team. Follow the instructions on the screen

create team

Input "invite team" followed by the name of the user that you want to invite to the team.

invite team abrams

The invited user can accept the invitation by typing "invitations" and following the instructions

invitations

Radio

You can listen in on audio streams or files.

Input "radio list" to see available streams and files. Input "radio on" followed by the ID of the file (the ID is within [] on each row). Input "radio off" to turn the audio off.

radio list

radio on ra1

radio off

Opening the Ark

LANTERNs

There will be multiple physical LANTERNs that will help the different factions open the Ark. The signal from these nodes can be amplified or dampened with the hacklantern command.

hacklantern allows anyone with access to the terminal to be part of the hunt for the Ark and decide the victor.

Clicking on LANTERN in the top menu or typing "lantern on" will show you information about the points. Clicking on the status windows or typing "lantern off" will hide it.

lantern on

lantern off

hacklantern usage

You start it up by running hacklantern. The goal of the command is to find a user and password combination that has access to the lantern. Start by selecting which station you want to access

You should now see a user name followed by several password hints. You will have to remember these, as they give clues as to which password is the correct one

The two next steps will show you blobs of text. You will have to use the hints from the user to figure out which of the words in the blobs is the correct password. The correct password will be shown in BOTH blobs

You will now be asked to input the correct user and password and choose if you want to amplify or dampen the signal to the lantern. Congratulations! You have just made a new friend/enemy!

You see two users, user1 and user2. user1 has the hint "password starts with pi." user2 has the hints "password starts with pi" and "password ends with za." You look through the memory dumps and find the word "pilot" in one of the memory dump. Even though it fits with the hint for user1 it was not in both memory dumps and is thus not the correct password. You find the word "pizza" in both memory dumps. The hints from user2 fits with the word, thus you choose to input user2 and "pizza" as the correct user and password. Yay! It worked!

The hunt

There are 4 rooms that you can follow, named lantern1, lantern2, lantern3, and lantern4. They correspond to each LANTERN in the game.

The rooms will receive notifications each time a person amplifies/dampens the signal. If you follow the rooms you will be able to quickly see who your new allies/enemies are.

Settings and requirements

Data usage

The application is very lightweight. The map parts are downloaded from Google when you scroll around and zoom in/out. It drains as much as visits to Google Maps do, which isn't much.

Using the radio command will drain more, as it is a continual audio stream. The amount depends on the bitrate.

Tracking

The application uses your location to add to the game map and allow others to see it. You should see a small pop-up asking you to allow it to access your location information. Your location will only be tracked when you are using the application and only your latest location will be stored and sent to all other users connected

This is part of the game. It will be a lot more fun for both you and other players if you allow it to track you while using the application.

Partially obstructing frame

Is your screen partially obstructed by a frame? Run "settings thinnerview on" to add some margin to the top, left and right sides. Running the same command but with off will remove the margin

Not using a mouse/touch interface?

Run "settings hiddencursor on" in the terminal to hide the cursor. Run "settings hiddenmenu on" to hide the clickable top menu. Run the same commands but with off to show the cursor/menu.

Desktop/Laptop

It works in Chrome. It might work in Firefox. Full screen does not work in Safari.

Full screen

Click on any text or use the shortcut ctrl+u. It should automatically expand to full screen.

Raspberry Pi

Iceweasel is recommended as a browser replacement for Chrome

Apple iOS

The application has been tested on iOS 7 and higher, on iPad and iPhone. Use Safari.

Full screen

Safari does not support full screen through normal means. You have to visit the site in Safari, add it to your home screen and run it from there. See this guide on Lifehacker on how to add it to your home screen

Android

The application has been tested on different version of Android 4, 5 and 6. It will not work on Android 2. Use Chrome.

Full screen

Click on any text. It should automatically expand to full screen.
